DNS Server

Dieser muss auf 127.0.0.1 oder ::1 eingestellt sein. Nur als Admin möglich.

Passt
Ändern

Fallback DNS Server

Falls der DNS-Proxy nicht läuft werden die DNS-Querys direkt an den Fallback DNS Server gesendet. Dadurch werden aber keine mehr gefiltert.

Schnittstellenname

Wähle deine Schnittstelle aus.

Internet Protocol Version

Die Version wird automatisch ausgewählt. Bei einer Änderung kann es sein, dass der DNSProxy nicht mehr funktioniert.

Starten mit Systemstart

DNSProxy automatisch mit dem Systemstart starten.

DNS-Serverkonfiguration zurücksetzen

DNS-Server werden von Windows zurückgesetzt. DNSProxy funktioniert dann nicht mehr.
